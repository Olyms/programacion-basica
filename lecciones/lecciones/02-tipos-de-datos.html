02. Tipos de datos

En esta lección aprenderás los tipos de datos más comunes usados en programación.

¿Qué es un tipo de dato?

Un tipo de dato indica qué clase de información guarda una variable, por ejemplo números, texto o valores lógicos.

Tipos de datos básicos

  • int: números enteros (1, 10, -5)
  • float: números decimales (3.14, 19.99)
  • string: texto ("Hola", "Juan")
  • bool: verdadero o falso (True / False)

Ejemplo en código (Python)

precio = 19.99
cantidad = 3
producto = "Agua"
hay_stock = True

total = precio * cantidad

print(producto)
print("Total:", total)
print("¿Hay stock?", hay_stock)

Explicación

  • precio es un número decimal (float)
  • cantidad es un número entero (int)
  • producto es texto (string)
  • hay_stock es verdadero o falso (bool)

Error común

Olvidar las comillas en los textos:
❌ producto = Agua
✅ producto = "Agua"
